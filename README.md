## 📝 About the Project

This is a lightweight and efficient **web-based appointment booking system** designed for dental clinics. It allows patients to easily schedule appointments online while enabling clinics to manage bookings seamlessly.

Built using **Python (standard library)** and **HTML, CSS, JavaScript**, the project requires no external dependencies.

---

## 🚀 Features

* 🦷 Online appointment booking form
* 📅 Date and time selection
* 📞 Indian phone number validation
* ✅ Success message after booking
* 💾 JSON-based data storage
* 📋 Admin panel to view appointments
* ⚡ Fast and lightweight backend

---

## 🛠️ Tech Stack

* **Backend:** Python (http.server)
* **Frontend:** HTML, CSS, JavaScript
* **Database:** JSON
